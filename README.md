# Лабораторные работы по САКОДу

## Введение

Данный репозиторий по сути своей является "общественным достоянием". Весь код изначально пишется **для себя**. Разрешается брать отсюда все, и изменять как душе угодно. Полный open-source. Наслаждайтесь.

## Установка

Для того, чтобы пользоваться материалами репозитория, удобнее всего клонировать его на своё устройство. Для этого необходимо установить git на своё устройство (в интернете есть кучу инструкций как сделать это для Виндовс) и прописать в командую строку следующую команду:
```
git clone https://github.com/Qurclinc/SACOD-LBs
```

Создастся папка, содержащая все файлы.

## Примечание

Все исполняемые файлы компилировались на Линуксе, и скорее всего у вас не запустятся. Так как я писал в vscode и компилировал через консольку, то вы можете сделать точно так же (при наличии установленного g++/gcc конечно же) с помощью команды:
```
g++ main.cpp -o main.exe
```
и запустить созданный файл. Но учитывая что многие пользуются Visual Studio - то можно просто добавлять файлы в своё решение. Как сказано в самом начале - вы вольны делать как душе угодно.
